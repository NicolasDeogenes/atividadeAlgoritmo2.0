Este projeto implementa dois sistemas de gerenciamento de documentos, 
simulando cenários reais de uma gráfica rápida, utilizando as estruturas 
de dados **Fila (Queue)** e **Pilha (Stack)**. O objetivo é demonstrar a 
aplicação prática de conceitos de Estruturas de Dados como FIFO (First-In, First-Out) 
e LIFO (Last-In, First-Out).

---

## Estrutura do Projeto

O projeto está organizado em pacotes Java, refletindo as diferentes responsabilidades 
de cada classe.

Para realizar o teste de Impressão execute o arquivo SistemaImpressora.java, já para o 
teste de Reimpressão execute o arquivo SistemaReimpressao.java.
